LunarSystem

A PHP/MySQL CMS that models content as RDF triples and renders through XSLT — extended into an RDF-native Semantic Web CMS.

This is the project's GitHub Pages site. It publishes the vocabulary; the application itself is a study artifact meant to run on localhost.

  • Ontology — the luna: vocabulary (Turtle)
  • Repository — source code
  • Documentation — architecture, RDF model, linked data, security
